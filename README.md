# MyLove

مشروع ASP.NET Core MVC جاهز للنشر المجاني على Render.

## أسرع طريقة للنشر

1. ارفع المشروع إلى GitHub.
2. ادخل إلى [Render](https://render.com/) وسجّل بحسابك.
3. اختر `New` ثم `Blueprint`.
4. اربط Repository الذي رفعت عليه المشروع.
5. Render سيقرأ ملف `render.yaml` ويجهز خدمة مجانية تلقائيًا.
6. بعد انتهاء الـ deploy ستحصل على رابط عام بالشكل:

`https://mylove.onrender.com`

إذا كان الاسم مستخدمًا، Render سيعطيك اسمًا قريبًا منه.

## ملاحظات

- الخطة المجانية على Render تعمل، لكن الخدمة قد تنام بعد 15 دقيقة بدون زيارات، وأول فتح بعدها قد يتأخر قرابة دقيقة.
- المشروع الآن يقرأ `PORT` من البيئة، لذلك مناسب للتشغيل على Render.
